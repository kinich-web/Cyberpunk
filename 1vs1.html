📋 Общее описание

Программная система предназначена для автоматизации рабочего места сотрудника методического отдела института. Обеспечивает централизованный учёт специальностей, кафедр и дисциплин, а также формирование аналитической отчётности.

Технологии: C#, .NET, Windows Forms (WPF), PostgreSQL, LINQ

🏠 Главное окно

После запуска открывается главное меню с четырьмя кнопками-заглушками для перехода к разделам:

  • Специальности — управление справочником специальностей
  • Кафедры — управление справочником кафедр
  • Дисциплины — управление справочником дисциплин
  • LINQ Запросы — выполнение аналитических отчётов

В правом нижнем углу расположена неоновая вывеска ❓ СПРАВКА — нажмите для открытия данного руководства.

📚 Специальности

Справочник содержит информацию о направлениях подготовки. Таблица включает столбцы:

  • ID — уникальный идентификатор
  • Код — код специальности (например, 09.03.01)
  • Название — полное наименование
  • Квалификация — присваиваемая степень (Бакалавр, Специалист)
  • Форма обучения — Дневная / Вечерняя / Заочная

Действия: Добавить, Изменить, Удалить, Сохранить в PostgreSQL.

🏛️ Кафедры

Справочник содержит информацию о структурных подразделениях. Таблица включает столбцы:

  • Название — полное наименование кафедры
  • Факультет — факультетская принадлежность
  • Телефон — контактный номер

Особенность: При добавлении/редактировании кафедры можно выбрать несколько специальностей из списка, устанавливая связь между кафедрой и направлениями подготовки.

📖 Дисциплины

Справочник содержит информацию об учебных дисциплинах. Таблица включает столбцы:

  • Название — наименование дисциплины
  • Семестр — номер семестра (1–8)
  • Часы — общее количество часов
  • Структура — Лекции / Семинары / Лабораторные
  • Отчетность — Зачет / Экзамен / Курсовая работа
  • Кафедра — закреплённая кафедра

Особенность: При добавлении дисциплины можно выбрать несколько специальностей (множественный выбор), для которых она читается.

🔍 LINQ Запросы

Окно содержит 6 аналитических запросов для формирования отчётности:

№ Название запроса Описание
1 Дисциплины по специальности Список дисциплин для выбранной специальности
2 Часы лабораторных по специальности Подсчёт лабораторных часов по специальности
3 Семестровая нагрузка кафедры Распределение нагрузки кафедры по семестрам
4 Дисциплины кафедры Перечень дисциплин, закреплённых за кафедрой
5 Мин/Макс продолжительность Дисциплины с min и max количеством часов
6 Экзамены и зачеты по кафедрам Статистика видов отчётности по кафедрам

💾 Хранение данных

Все данные сохраняются в базе данных PostgreSQL.

  • Таблицы: specialties, departments, disciplines
  • JSONB-поля: specialty_ids (кафедры), structure (дисциплины)
  • Целостность: транзакции, каскадное удаление

При ошибках подключения программа выводит информативное сообщение.

🎨 Киберпанк-интерфейс

Интерфейс выполнен в стиле киберпанк с использованием:

  • Неоновых цветов — голубой, розовый, зелёный, жёлтый
  • Эффектов свечения — DropShadowEffect
  • Моноширинного шрифта — Consolas
  • Тёмного фона — с сеткой и матричными элементами

⚠️ Обработка ошибок

Система устойчива к некорректным действиям пользователя и сбоям:

  • Валидация обязательных полей в диалоговых окнах
  • Перехват исключений при работе с PostgreSQL
  • Подтверждение удаления записей
  • Информативные сообщения об ошибках

🖥️ Системные требования

  • ОС: Windows 7/8/10/11
  • Платформа: .NET 4.7 / .NET 6+
  • СУБД: PostgreSQL 12+
  • ОЗУ: от 512 МБ
